### Multi-Tenancy Design
- Database-level: Every table has `tenant_id`
- Query-level: All repositories filter by tenant
- Middleware-level: JWT validates tenant access
- Import-level: Routes verify tenant ownership
- Document-level: Files tagged with tenant_id for isolation

### RAG System
1. Query → Embedding (LLM)
2. Vector search with metadata filters
3. Relevance ranking (confidence scoring)
4. Context building from retrieved docs
5. LLM prompt with grounding instruction
6. Hallucination mitigation checks
7. Citation of sources

## 💡 KEY ALGORITHMS

### Skill Gap Calculation
```
Gap = RequiredLevel - CurrentLevel
DaysToClose = Gap * 30 / ExperienceFactor
(30 days base per level, adjusted by experience)
```

### Promotion Readiness Scoring
```
ReadinessScore = Σ(Skill_Score × Skill_Weight) / Total_Weight

Where:
  Skill_Score = (CurrentLevel / RequiredLevel) × 100
              × RecencyFactor(LastAssessmentDate)
```

### Risk Calculation
```
RiskScore = (AtRiskSkillPercentage × 0.6)
          + (ConcentrationRisk × 100 × 0.4)

AtRiskSkills = No assessment in 12+ months
ConcentrationRisk = Coefficient of variation of skill levels
```

### Analytics Aggregation
```
Team_Readiness = Σ Employee_Readiness / Team_Size
Department_Risk = Σ Employee_Risk / Department_Size
Promotion_Velocity = Promotions_Last_30_Days
```

## 📋 API REFERENCE

### Assessment Endpoints
```
POST   /api/assessments/:assessmentId/start
       Start assessment for employee

POST   /api/assessments/:resultId/submit
       Submit answers and get scoring

GET    /api/employees/:employeeId/assessments
       List all assessments for employee

GET    /api/assessments/:resultId
       Get assessment detail with feedback
```

### Skill Intelligence
```
GET    /api/employees/:employeeId/promotion-readiness/:roleLevel
       Analyze promotion readiness (returns readiness score, gaps, timeline)

GET    /api/employees/:employeeId/skill-risk
       Calculate skill risk (returns risk score, at-risk skills)

GET    /api/employees/:employeeId/skills
       Get employee skill profile
```

### Analytics
```
GET    /api/analytics/snapshot
       Complete snapshot (team readiness, risk clusters, promotion metrics)

GET    /api/analytics/team-readiness
       Team readiness by department

GET    /api/analytics/department-risks
       Department risk clusters

GET    /api/analytics/promotion-pipeline
       Promotion pipeline metrics
```

### RAG (Knowledge)
```
POST   /api/rag/query
       Execute RAG query with company knowledge grounding

POST   /api/rag/promotion-analysis/:employeeId
       Generate promotion analysis grounded in company docs

POST   /api/rag/incident-simulation/:skillId
       Generate incident guidance grounded in procedures
```
